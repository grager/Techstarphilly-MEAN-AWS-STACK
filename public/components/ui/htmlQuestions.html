1. doctype

Question: What is doctype? Why do u need it?
Anwser: doctype is an instruction to the browser to inform about the version of html document and how browser should render it.It ensures how element should be displayed on the page by most of the browser. And it also makes browser's life easier. otherwise, browser will guess and will go to quirks mode. Moreover, doctype is required to validate markup.

2. data-*

Question: What is the use of data- attribute?
Anwser: allow you to store extra information/ data in the DOM. u can write valid html with embedded private data. You can easily access data attribute by using javascript and hence a lot of libraries like knockout uses it.
